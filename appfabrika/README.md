# AppFabrika

BMAD metodolojisi ile urun gelistirme CLI araci.

## Kurulum

```bash
npm install -g appfabrika
```

## Kullanim

```bash
# Yeni proje baslat
appfabrika init

# BMAD workflow'u calistir
appfabrika run
```

## Gelistirme

```bash
# Bagimliliklari yukle
npm install

# Build
npm run build

# Watch mode
npm run dev

# Test
npm test
```
